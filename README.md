## Stack
- HTML5 · CSS3 · Vanilla JS — no frameworks, no build step
- Google Fonts: Inter + Playfair Display
- Deployed via GitHub Pages

## Pages
| File | Page |
|------|------|
| `index.html` | Home |
| `about.html` | About |
| `resume.html` | Résumé (live page) |
| `projects.html` | Projects |
| `contact.html` | Contact |
| `style.css` | Shared design system |

## Running locally
```bash
# Python (built into macOS/Linux)
python3 -m http.server 3000
# Then open http://localhost:3000
```

## Adding your resume PDF
Drop your PDF at `assets/resume.pdf` — the résumé page and nav button link to it automatically.

## Design system
All colors, type, and spacing are CSS custom properties in `style.css` under `:root`.
Dark background · Rose pink accent (#E8709A) · Playfair Display headings · Inter body.
